RileyRover - Robô com Garra e Sensor Ultrassônico

🤖 Sobre o Projeto
Este projeto consiste no desenvolvimento e programação de um robô móvel baseado no design RileyRover utilizando o kit LEGO Mindstorms EV3. O objetivo principal é a navegação para detecção, aproximação e captura de objetos através de uma garra motorizada.

⚙️ Hardware e Configuração
O robô foi construído com as seguintes especificações:
Unidade de Processamento: LEGO EV3 Intelligent Brick.
Atuadores de Locomoção: 2 Motores Grandes (Left Motor: Port B / Right Motor: Port C).
Atuador da Garra: 1 Motor Médio acoplado ao mecanismo de engrenagens para abertura e fechamento.
Sensor de Proximidade: Sensor Ultrassônico montado na parte frontal para medição de distância.
Base: RileyRover Base Design com Back Castor para estabilidade.

🚀 Funcionalidades
Navegação: O robô percorre o ambiente evitando obstáculos.
Detecção de Objetos: Utiliza o sensor ultrassônico para identificar objetos a uma distância pré-definida.
Captura (Gripper): Ao detectar um objeto, o robô se aproxima e aciona o motor médio para fechar a garra e segurar o item.
Transporte: Após a captura, o robô pode ser programado para levar o objeto a um local específico.
